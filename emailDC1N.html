Hello,

We plan to simplify Active Directory Domain Services (AD DS) management by ensuring that all the user accounts at the Paris office are placed in an organizational unit (OU) named Paris. The Paris OU must also contain a universal group for managing administrators at the Paris office.

A temporary contract user was recently hired to help manage the Paris office users. We plan to have the contractor use a generic account named ParisContractor that will be granted permissions to perform the intended management tasks.

This is your task:

You need to create the domain objects for the Paris office. The solution must meet the following requirements:

  • Create an OU named Paris that will contain a user object named ParisContractor and a universal group named Paris Admins.
  • ParisContractor must have an expiration date of December 31, 2027, and be a member of the Paris Admins group.
  • ParisContractor must be configured as a protected account.
  • The Paris Admins group must be able to manage only the passwords of users in the Paris OU.
